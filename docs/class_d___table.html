Detailed Description

The D_Table class Display tables in QTableWidget in the user interface.

Put a QTableWidget somewhere in the user interface and pass a pointer to it to (<D_Table>"::")n<set_TW>"("<(QTableWidget *TW)")" and changes made to the D_Table will be displayed in the UI
