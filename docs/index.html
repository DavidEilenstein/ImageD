ImageD  v0
ImageD is designed for rapid prototyping and implementing algorithms and solutions for science, research and teaching. ImageD is not intended to be used for industrial applications or high speed solutions. It is developed by David Eilenstein at GSI Helmholtz Centre for Heavy Ion Research.
ImageD Documentation Mainpage

Introduction

ImageD is an image processing software.

What ImageD is intended to do:

  • Graphical user interface
  • Easy to handle for beginner users
  • Lots of parametric functions for advanced users
  • Multidimension images and videos (up to 6D)
  • Rapid prototyping
  • Dynamic image processing chains
  • Interactive image analysis
  • Lots of plugins for biophysics applications at GSI Darmstadt

What ImageD is not intended to do:

  • Industrial applications like quality control
  • High high speed optimization

Installation

ImageD is published as OpenSource under the GPL3 licence. Its github repository can be found here.

Installation:

  1. Go to https://github.com/DavidEilenstein/ImageD/releases
  2. Select the version you want to dowload (click on the blue headline)
  3. Download the "ImageD_v<number>.zip"
  4. Unzip it in a folder of your choice
  5. Double click the ImageD.exe to run the program

Hint for users at GSI without admin permissions: The folder mentioned in 4. must but a subfolder of "C:\Users\<username>\AppData\Local\0_gsi_executables". Otherwise the program can only be executed with admin permissions or if it is unlocked by the IT.
